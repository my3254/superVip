# SuperVip Mobile Web

独立手机网页项目，适合在同一 Wi-Fi 下用 iPhone Safari 访问。

## 本地运行

```bash
npm run start
```

启动后终端会打印两个地址：

- `http://localhost:4173/`：电脑本地查看样式
- `http://你的局域网IP:4173/`：iPhone 连接同一 Wi-Fi 后访问

## 使用方式

1. 在手机 Safari 打开平台官网。
2. 复制视频页地址。
3. 回到 SuperVip Mobile Web 粘贴地址。
4. 选择解析线路，点击“开始解析”。

手机网页端不会嵌入平台官网页面；这是 iOS Safari 的浏览器能力限制，不是代码没有做。
